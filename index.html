Base Ecommerce

Se creo una plantilla basica para el futuro desarrolo de ecommerce's, desarrollada en java springboot con mysql utilizando arquitectura MVC y algunos patrones de diseño como el patro builder. Se uso una pasarela de pagos de MercadoPago, el manejo de usuarios se hace con auth0, al realizar el pedido se le envia un mail al cliente con la informacion del pedido me arrojo un gran desafio el problema de los tiempos de espera, y como manejarlos. Se penso una configuracion por dia de los tiempos de espera donde se pueda elegir un rango horario para cual configurarlos, toda superposicion de horarios y dias se chequea antes de dar de alta o editar una configuracion
